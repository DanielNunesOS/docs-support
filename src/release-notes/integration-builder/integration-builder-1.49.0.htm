Integration Builder 1.49.0

Released on Jan 26, 2024

New in Integration Builder 1.49.0

  • Added support for transactions to the MongoDB connectors created by the Integration Builder. (RCNT-4777)
  • Now, marking a Salesforce object as “read-only” will allow you to unselect most of its attributes. (RCNT-6831)
  • Now, marking a SharePoint list as “read-only” will allow you to unselect most of its fields. (RCNT-6833)

Bug Fixing

  • Fixed an error un-checking the ‘Access type' checkbox on the 'Add Objects' step of Salesforce integrations and on the 'Add Lists’ step of SharePoint integrations. (RCNT-6844)
  • Fixed an "Index out of bounds" error when setting custom Foreign Keys on Relational Database Integrations. (RCNT-6881)
  • Breaking change: we were using an attribute that could be modified as an identifier for SharePoint columns. We had to change it, so some columns that were previously selected might need to be selected again when you edit a SharePoint integration. The system will display a warning message when this happens. (RCNT-6907)
