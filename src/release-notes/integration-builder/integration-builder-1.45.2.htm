Integration Builder 1.45.2

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue where the database username would not be displayed in Integration Manager, for MySql and PostgreSQL advanced configurations. (RCNT-6279)
  • Fixed an issue where the dropdown to choose the database catalog wouldn’t be displayed when installing the MongoDB Integration Manager Plugin on a server with multiple database catalogs. (RCNT-6331)
  • Fixed an issue that caused POST, PUT, and DELETE requests to fail with a 403 forbidden error when the 'sap-client' parameter value wasn't the default. (RPM-4140) Extended Product Integrations Builder
  • Fixed an issue where Integration Manager requests to Integration Builder would fail when the platform server’s identity provider was LDAP or Active Directory. This issue blocked users from creating connections. (RPM-4162) Extended Product Integrations Builder
