Integration Builder 1.44.0

This is a preliminary version of the document.

New in Integration Builder 1.44.0

  • It's now possible to rename entities and attributes when creating SAP OData integrations. (RCNT-5371)
  • It's now possible to rename entities and attributes when creating MongoDB integrations. (RCNT-5372)

Bug Fixing

  • Fixed an issue that caused Salesforce development connection keys to take up more database storage space each time they were used. (RCNT-5897)
