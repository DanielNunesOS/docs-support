Integration Builder 1.43.1

Released on Feb 23, 2023

New in Integration Builder 1.43.1

  • Changed the description text in the MySQL integration card to show MySQL 8.0.28+ is supported on platform version 11.19.0 and later. (RCNT-5641)
  • Added text to clarify the certificate options when creating a Salesforce connection manually. (RCNT-5735)

Bug Fixing

  • Fixed an issue renewing the authorization of Salesforce development connections. (RCNT-4380)
  • To prevent the Salesforce connector's generation from failing when there are too many picklists, now, an action is generated to sync each object picklist. (RPM-3640) Extended Product Integrations Builder
  • Fixed an issue getting files with names that contain special characters (# and %). (RPM-3656) Extended Product Integrations Builder
  • Fixed a crash caused by specifications with recursive references. (RPM-3678) Extended Product Integrations Builder
