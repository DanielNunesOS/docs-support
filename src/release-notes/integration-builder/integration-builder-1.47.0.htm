Integration Builder 1.47.0

Released on Nov 30, 2023

New in Integration Builder 1.47.0

  • Added the ability to manually set the description of external database entities. (RCNT-5101)
  • Updated the user interface to state that MongoDB versions 6.x and 7.0 are supported. (RCNT-6692)

Bug Fixing

  • Fixed an issue where the user interface would incorrectly state the solution pack for Salesforce integrations included the SAP OData Integration Manager Plugin. (RCNT-6660)
  • Fixed an issue where the layout of the "Rename entity" modal for Relational Database integrations would break if the table name was very long. (RCNT-6724)
  • Fixed an issue where it was possible to republish an integration that had been edited in Service Studio (which would overwrite any changes made using Service Studio). (RCNT-6742)
