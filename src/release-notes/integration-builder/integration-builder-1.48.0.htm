Integration Builder 1.48.0

This is a preliminary version of the document.

New in Integration Builder 1.48.0

  • It’s now possible to mark attributes of relational database entities as being references. While this doesn’t have any effect on the runtime of consumer applications, it allows Service Studio to give developers better suggestions when they are using those entities. (RCNT-4638)
  • Fixed an issue that caused some read-only attributes of Salesforce objects to be marked as mandatory. (RCNT-6688)

Bug Fixing

  • Fixed an issue where text would overflow some tooltips. (RCNT-6740)
  • Fixed an issue where Salesforce and Relational Database integrations could be flagged as having unpublished changes when they didn’t, (RCNT-6803)
