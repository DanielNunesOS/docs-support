Architecture Dashboard 1.18

This is a preliminary version of the document.

New in Architecture Dashboard 1.18

  • Improved the"Non-optimized Local Storage" pattern to avoid raising findings when the Local storage is referenced from other modules. (RADRT-2012)
  • Revamped the Overview Dashboard Variation chart to present the technical debt variation vs. the number of findings in a more clear way. (RADRT-2018)
  • The "Dynamic Inline Parameter pattern" was discontinued, due to the redundancy with the "SQL Injection" pattern. (RADRT-2094)

Bug Fixing

  • Improved the "Incorrect offline sync method" to avoid raising findings when the "TriggerOfflineDataSync" method is being used. (RADRT-1765)
  • Fixed an issue on the "Inadequate data preparation" pattern that was failing the pattern analysis when there were disabled nodes. (RADRT-2099)
