AI Mentor Studio 1.33

Released on Oct 10, 2024

New in AI Mentor Studio 1.33

  • [AI Mentor Studio API] Optimized the way how the API retrieves the latest technical debt information reducing the response time of calls when requesting the current technical debt. (RADRT-2929)
  • [AI Mentor Studio API] The API "ArchitectureDashboardAPI" will be removed after December 31, 2024. Customers still using this API must change the address of their integrations to "AIMentorStudioAPI", which has the exact same behavior and functionality. (RADRT-2951)
  • Now the Teams and User drop-down filters are also searchable. (RADRT-2966)
  • Updated missing Japanese translations in previously untranslated patterns. (RADRT-3179)

Bug Fixing

  • Fixed an issue on the Duplicated Code pattern that could rarely cause findings to be added to an existing pattern instead of creating a separate one. Affected patterns will be fixed automatically after a synchronization. (RADRT-2912)
  • Fixed an issue where AI Mentor Studio would automatically remove "www." from the environment's URL that was provided when attempting to login using the IT User account. (RADRT-3129)
  • [AI Mentor Studio API] Fixed an issue from a recent change that caused the API to return incorrect information for application related findings, where instead of returning all the patterns with application findings, the API was returning multiple copies of the same pattern. Module related findings were not impacted by this issue. (RADRT-3181)
