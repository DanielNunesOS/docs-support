AI Mentor Studio 1.25

Released on May 31, 2023

New in AI Mentor Studio 1.25

  • [AI Mentor Studio Probe 4.3] Released version 4.3 of the probes which includes a new mechanism to support IT User authentication. (RADRT-2384)
  • [AI Mentor Studio Probe 4.3] AI Mentor Studio probes are now only configurable by users with Full Control permissions in the infrastructure. (RADRT-2400)
  • Now, customers can register their infrastructure using both the OutSystems account and IT user authentication. (RADRT-2523)

Bug Fixing

  • [AI Mentor Studio Probe 4.3] Changed the feedback messages with links to alert messages. (RADRT-2483)
  • Fixed a timeout issue that was happening in certain scenarios when exporting the findings report to Excel. (RADRT-2506)
  • Fix an issue that was causing some inconsistencies between the number of findings shown in the report area and the Overview Dashboard. (RADRT-2529)
  • Fixed an issue where findings from the “Avoid setting Registered Roles to access Screens” pattern were wrongly reopened due to losing the resolve information. (RADRT-2530)
  • Fixed an issue in the AI Mentor Studio API where the “TechnicalDebt_Team” method was giving timeout in factories with many teams. (RPM-3966) AI Mentor Studio
