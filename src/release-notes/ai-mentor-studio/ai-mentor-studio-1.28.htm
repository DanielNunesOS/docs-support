AI Mentor Studio 1.28

Released on Nov 23, 2023

New in AI Mentor Studio 1.28

  • Enhanced "Open in Service Studio" link coverage by ensuring that if an element is not permitted to be opened, the navigation will default to the first top parent that is allowed to open in Service Studio. (RADRT-2635)
  • Now, users can leverage the information provided by the AI Mentor Studio API to build custom URLs that redirect to the Report page filtered by the parameters “TeamExternalId” and/or “ApplicationGuid”. (RADRT-2652)
  • Updated the user interaction in the element tree view within the findings list to switch from click to hover. Additionally, adjusted the tooltip position in the element tree view to be displayed on the right. (RADRT-2654)
  • Improved the experience of the apps/modules filter selection on the report’s page. (RADRT-2732)
  • Improved the performance of the code analysis process while analyzing findings. (RADRT-2738)

Bug Fixing

  • Fixed an issue where, after leaving an AI Mentor Studio tab open for an extended period with a selected Team, the screen refresh incorrectly displayed the message 'This team has no applications yet.' (RADRT-2671)
  • Fixed an issue that was blocking customers from using external IdP providers like PingFederate. (RADRT-2714)
