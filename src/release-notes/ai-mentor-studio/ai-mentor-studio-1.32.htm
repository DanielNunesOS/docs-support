AI Mentor Studio 1.32

Released on Apr 22, 2024

New in AI Mentor Studio 1.32

  • [AI Mentor Studio API] The method TechnicalDebt_Application now returns an error when the provided offset exceeds the number of available records. (RADRT-2877)
  • Updated missing Japanese translations across AI Mentor Studio. (RADRT-2878)
  • Due to poor adoption, the Mini-Hub menu with navigation links for documentation and other product parts available in the upper right corner of all screens, has been removed from AI Mentor Studio. (RADRT-2887)

Bug Fixing

  • [AI Mentor Studio API] Fixed an issue on the method TechnicalDebt_Application where the value returned by TotalResults could be lower than the actual number of records available under certain scenarios. (RADRT-2875)
  • Fixed an issue on the Report screen that could cause a timeout for customers with larger factories in some circumstances. (RADRT-2899)
