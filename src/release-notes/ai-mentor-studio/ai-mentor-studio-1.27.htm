AI Mentor Studio 1.27

Released on Sep 25, 2023

New in AI Mentor Studio 1.27

  • Improved performance and detection of the code patterns: “Server requests in client events”, “Long Server Requests Timeout” and “Inadequate data preparation”. (RADRT-2492)
  • [AI Mentor Studio Probe 5.0] Released version 5.0 of the probes, which addresses the vulnerability RPM-4141, ensuring the continued security and reliability of AI Mentor Studio. (RADRT-2594)
  • Fixed an issue where, in some scenarios, users would see an error message “User not found” after logging in. (RADRT-2596)
  • Fixed a security vulnerability with CVSSv3.1 score of 8.6. Check how to keep your system protected at the security bulletin. (RPM-4141) AI Mentor Studio
