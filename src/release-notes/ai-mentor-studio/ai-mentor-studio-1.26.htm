AI Mentor Studio 1.26

Released on Sep 22, 2023

New in AI Mentor Studio 1.26

  • Added a new security pattern “Insecure Usage of GetUserId in client Block parameters” which is focused on avoiding passing identity information from the client side to the server side as parameters. (RADRT-2544)
  • Added a new maintainability pattern “Unused Aggregate or SQL Query” which helps to identify unused data queries. (RADRT-2545)
  • Improved the Duplicated Code findings in the report area, to show the findings' count instead of the aggregators' count. (RADRT-2563)

Bug Fixing

  • Category filter is not persistent (between sessions) (RADRT-2616)
  • Fixed an issue on AI Mentor Studio API where invoking the TechnicalDebt_Application method while data is being refreshed could result in incomplete data being returned. Now, status code 503 will be returned in this scenario, along with the retry-after header which indicates how many seconds to wait before retrying. (RPM-4135) AI Mentor Studio
