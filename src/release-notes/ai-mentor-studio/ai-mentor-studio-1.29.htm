AI Mentor Studio 1.29

Released on Dec 18, 2023

New in AI Mentor Studio 1.29

  • Enhanced the user interface on the Report findings list, allowing users to easily identify the module associated with each finding directly within the list. (RADRT-2655)
  • Performance improvements on the synchronization process. (RADRT-2742)

Bug Fixing

  • Fixed an issue in the maintenance area, in the code patterns list, where some patterns appeared duplicated when navigating through pages. (RADRT-2730)
  • Fixed an issue on the pattern “Long undocumented flow” where some larger flows were not being detected properly. This may cause a slight increase in the number of findings. (RADRT-2756)
