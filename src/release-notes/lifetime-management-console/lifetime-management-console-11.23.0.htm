LifeTime Management Console 11.23.0

Released on Jun 12, 2024

New in LifeTime Management Console 11.23.0

  • There is a new action on LifeTimeSDK (GetUserAPIToken) that allows the user currently logged in into LifeTime to obtain an authentication token valid for 5 minutes. This token can be used for any LifeTime API call that the user has access to, working similarly to a service account authentication token. (R11CT-3060)
  • It is now possible to refresh the authentication token of Service Accounts using LifeTime API v2. The new method /auth/tokens/ allows to refresh the authentication token through the api for the Service Account calling the api. The old token is no longer valid and the returned token should be used for any future requests. Previously, this operation was only possible by using LifeTime UI. (R11CT-3078)

Bug Fixing

  • Fixed a bug in the warning/error link in the stage steps of a deployment plan. Now, when clicking on these links it redirects to the corresponding staging message. Inspired by Evert Van Der Zalm's feedback. (R11CT-3019)
  • Fixed an issue that allowed users to force a password that doesn't meet complexity requirements when updating a user. (RPM-4782) Application Lifecycle LifeTime
  • Previously, when the LifeTimeAPI was used to retrieve the list of conflicts in a deployment, it only identified one conflict per application. We've improved the LifeTimeAPI to now fetch all conflicts for all applications within a deployment. (RPM-4799) Application Lifecycle LifeTime
  • Fixed an issue that slowed the load of deployment messages in the Staging Progress screen during the deployment. In deployments with a high number of messages created in a short timespan, the page should now load new messages much faster. (RPM-4928) Application Lifecycle LifeTime
  • Fixed an issue where the Lifetime API returned the incorrect staging status when the status was set to "forced finish." Now, when the staging is forced to finish, the API will return the correct staging status. (RPM-4930) Application Lifecycle LifeTime
