LifeTime Management Console 11.16.5

Released on Mar 20, 2023

Bug Fixing

  • Fixed an issue that caused scroll to be missing after opening a popup. Now scrolls are never removed and popups are always locked in place. (R11CT-1986)
  • Improved the synchronization of service account user permissions across the registered environments in LifeTime. When the user's permissions are not well synchronized, some information becomes outdated in LifeTime. (RPM-3693) Application Lifecycle LifeTime
  • Fixed a bug that would make page scroll become disabled after toggling "Enhanced security for Remoting requests". The fix is included in R11CT-1986. (RPM-3767) Application Lifecycle LifeTime
  • Fixed an issue that caused log pollution when parsing a JSON. This would happen when authenticating IT users through ADAuthentication while using the pattern Domain\UserName. (RPM-650) Application Runtime Logging
  • Fixed an error that prevented deleting roles when Service Accounts were associated. Now when a role is deleted it will prompt to replace the role on all Users and Service Accounts. (RPM-698) Application Lifecycle Users
  • Fixed a security vulnerability that allowed unauthorized users to access application permissions data. CVSSv3.1 score 4.3 (Medium). (RPM-3702) Application Lifecycle LifeTime
