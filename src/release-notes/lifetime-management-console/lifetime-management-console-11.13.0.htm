LifeTime Management Console 11.13.0

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue that caused a deployment to fail when the saved deployment plan included an app that was deleted after plan was saved. (R11CT-913) (R11CT-913)
  • Now during a staging a process applications are created before modules. (RPM-2262)
  • Fixed an issue that caused LifeTime sync to be blocked due to activity timeout when there is a large number of applications for an environment and there is a high level of latency between the environment and lifetime (RPM-2308)
  • Fixed a bug that allowed the user to add more apps to a staging that was already running (RPM-2522)
  • Fixed an issue that was removing the template key from applications during staging, causing those applications to not be deployed. (RPM-2555)
