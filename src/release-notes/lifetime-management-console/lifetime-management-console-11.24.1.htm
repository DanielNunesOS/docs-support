LifeTime Management Console 11.24.1

Released on Aug 07, 2024

New in LifeTime Management Console 11.24.1

  • The environments screen in LifeTime now has a dedicated "Network" section in the environment details panel where customers can find the already existing options to manage the environment certificate and its secure endpoints. This change affects only Cloud customers. (R11CT-3224)

Bug Fixing

  • Fixed an issue that would cause the environment filters' dropdown to be hidden when a filter with only one environment was selected even if there were more environments in the infrastructure. Now, the dropdown is correctly displayed when there is more than one visible environment in the infrastructure and is hidden otherwise. (R11CT-3282)
  • Fixed an issue that caused a database error when deleting application versions using LifeTime API DELETE /applications/{ApplicationKey}/versions/{VersionKey}. This occurred when the application version being deleted had references to a deleted environment. (RPM-3861) Application Lifecycle LifeTime
  • When configuring an OpenId Connect Provider, a new message will be displayed immediately before activating the provider, warning of mandatory manual steps that must be taken. Failing to do these mandatory steps will have adverse effects, such as Service Studio users being unable to maintain their sessions authenticated. (RPM-4890) Application Lifecycle Users
  • Fixed an issue when disabling maintenance mode for a given environment that would not propagate OpenId Connect configurations that had changed while the environment was in maintenance mode. (RPM-5076) Application Lifecycle LifeTime
