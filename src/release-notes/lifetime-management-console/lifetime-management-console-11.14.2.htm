LifeTime Management Console 11.14.2

This is a preliminary version of the document.

New in LifeTime Management Console 11.14.2

  • With IDP activated, Break glass users will be able to login to Lifetime using a new entry point /InternalLogin.aspx (R11CT-1392)
  • With IDP flow activated, Now Password fields on New/Edit User screen will be hidden for Default users. (R11CT-1397)
  • LifeTime now uses Service Center login screen for the authentication flow. (R11CT-1493)

Bug Fixing

  • Fixed a timeout issue in LifeTime's synchronization process when there was a large number of applications for an environment or high latency between an environment and LifeTime. (RPM-2308) Application Lifecycle LifeTime
  • Fixed an issue that allowed users to add applications to a deployment plan that was already running. (RPM-2522) Application Lifecycle
  • Fixed a bug that was allowing to tag an application without a module (RPM-2538) Application Lifecycle LifeTime
  • LifeTime's Installation Checklist is now consistent with Platform Server's instructions. Installation steps have been revised and updated. (RPM-2558) Infrastructure Management Platform Installation Checklist
  • Fixed an issue preventing an internal field 'TemplateKey' from being lost when staging applications between environments. (RPM-2725) Application Lifecycle LifeTime
  • LifeTime's Installation Checklist now has clear instructions on how to create the LifeTime application pool. (RPM-2731) Application Lifecycle LifeTime
  • Updated installation information about Oracle 19c. (RPM-944) Platform Installation Checklist
