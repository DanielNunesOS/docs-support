LifeTime Management Console 11.19.0

Released on Nov 22, 2023

Bug Fixing

  • For scenarios where site property values are changed, staging summary progress messages now include the module name to which the site properties belong. (R11CT-2573)
  • Fixed an issue in LifeTime's checklist. The Microsoft Visual C++ 2015-2022 Redistributable (x64) is now included in the installation requirements. (RPM-4399) Infrastructure Management Platform Installation Checklist
  • Fixed an issue that prevented users from seeing the full application name or user roles' name on some LifeTime screens. This issue occurred when the name was too long and would be concatenated and displayed with an ellipsis. Now, the space for these attributes has been increased, allowing for full visibility. (RPM-4401) Application Lifecycle LifeTime
  • Fixed circular dependencies in LifeTime solution that would prevent the proper installation of LifeTimeSDK in development environments. (RPM-4504) Application Lifecycle LifeTime
