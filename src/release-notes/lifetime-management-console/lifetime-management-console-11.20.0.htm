LifeTime Management Console 11.20.0

Released on Dec 20, 2023

New in LifeTime Management Console 11.20.0

  • Platform Server version updated to 11.24.0. (R11CT-2671)
  • Site properties can now be defined as a secret, which will ensure their value is stored and encrypted in the database. This requires Platform Server 11.25 or higher and ServiceStudio 11.54.35 or higher. You can learn more about the feature here. (R11CT-2672)

Note

  • LifeTime 11.20.0.3199 fails to recognize when secret site properties are used in applications. This makes it possible for secret site properties to be configured, as part of the deployment plan configuration process, and for the respective applications to be deployed to the target environment without any apparent problem. The problem manifests at application runtime, with an error message of "Unable to decrypt content" when accessing the secret site property value. This behaviour has been corrected in LifeTime 11.20.1. To work-around the problem caused by LifeTime 11.20.0.3199, it is necessary to go to the ServiceCenter of the affected environment and to set manually a new value for the secret site property. Re-publishing the affected module does not fix the problem.
