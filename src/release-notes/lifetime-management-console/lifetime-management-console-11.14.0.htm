LifeTime Management Console 11.14.0

This is a preliminary version of the document.

Bug Fixing

  • When a Service module is renamed and published, there's a compilation message about potential incompatible consumers and these consumers are now considered outdated applications in LifeTime (RPM-1665)
  • Fixed a timeout issue in LifeTime's synchronization process when there was a large number of applications for an environment or high latency between an environment and LifeTime. (RPM-2308)
  • Fixed an issue that was allowing the team to be added to the Service center application when using Lifetime API (RPM-2393)
  • Fixed an issue that was causing incorrect URL generation for app download on the cloud infrastructure when Lifetime API was called to get the host URL. (RPM-2587)
  • Fixed an issue that caused errors during site properties synchronizations, leaving LifeTime outdated and impacting the feature site properties configuration during deployment (RPM-2716)
  • Reduced error log pollution when synchronizing deleted modules. (RPM-2738)
  • Reduced error log pollution when synchronizing deleted modules. (RPM-2740)
