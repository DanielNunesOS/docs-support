LifeTime Management Console 11.17.1

This is a preliminary version of the document.

New in LifeTime Management Console 11.17.1

  • Updated UI on LifeTimeCloudConnect that would provide incorrect/inconsistent information regarding database-related operations. Now, LifeTimeCloudConnect provides consistent information. (R11CT-2053)
  • Improved the user experience in the list of Application Versions on the Deployment Plan page by making it clearer which was the previous application version and which application version was updated to the target environment. (R11CT-2206)

Bug Fixing

  • Fixed an issue that caused a timeout error when the Open ID authentication provider is activated in LifeTime. This occurred mainly when the infrastructure is large or the environments are spread out in different locations. Now, it is possible to configure the timeout value based on infrastructure needs. (RPM-3908) Application Lifecycle Users
  • Fixed an issue that caused an internal error when accessing the "Application_History" screen for applications without versions. (RPM-3996) Application Lifecycle LifeTime
