LifeTime Management Console 11.14.1

This is a preliminary version of the document.

New in LifeTime Management Console 11.14.1

  • Updated the Platform Server version to 11.17.0. (R11CT-1420)

Bug Fixing

  • Added a feature toggle to skip a full synchronization process, which was responsible for deadlocks when staging a large number of applications. (RPM-2052) Application Lifecycle LifeTime
  • Fixed an issue that caused outdated data to be retrieved in the LifeTimeSDK APIs when checking information for users that haven't logged in at the moment the data is being read. (RPM-2745) Application Lifecycle LifeTime
  • There is no longer an error shown during staging validations when a user doesn't have permission to download a module included in the staging plan. (RPM-2755) Application Lifecycle LifeTime
