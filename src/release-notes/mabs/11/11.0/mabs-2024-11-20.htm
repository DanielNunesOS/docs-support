MABS 2024-11-20

Released on Nov 20, 2024

New

  • MABS 11 now uses Node 20.
  • Updated the Native Shell to target the latest iOS 18 SDK. (RDMR-251)
  • Updated the Native Shell to target the latest Android 15 SDK. (RDMR-251)

Breaking Changes

  • Changed the AddUploadWidgetPermissions property default value from true to false. The Android permissions associated with the upload widget are no longer added out of the box. (RDMR-299)
  • Changed the StatusBarOverlaysWebView property default value from false to true. (RDMR-331)
  • Added the StatusBarStyle property default value as default. (RDMR-331)

Known Issues

  • Applications with some special characters in the name might fail to build on iOS, example found is the equals sign (=) (RNMT-6887).
