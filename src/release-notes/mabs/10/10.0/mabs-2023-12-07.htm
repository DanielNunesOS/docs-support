MABS 2023-12-07

Released on Dec 07, 2023

New

  • We added the ability for plugins to provide a specific error message by making use of the OUTSYSTEMS_PLUGIN_ERROR: pattern. Check the documentation for more details. (RNMT-6353)

Bug Fixing

  • We fixed a scenario where an invalid value on extensibility configurations was resulting in OS-MABS-CNF-50000 error instead of OS-MABS-CNF-40005. (RNMT-6384)
