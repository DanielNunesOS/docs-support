Cross-platform Service Studio 11.54.16

Released on Jul 10, 2023

New in Cross-platform Service Studio 11.54.16

  • Now, when uploading a file to consume a SOAP or REST service, the file upload window will default to the last folder used for uploading files. (RO11IT-301)

Bug Fixing

  • Fixed an error when consuming SOAP Web Services containing a xs:complexType with an xs:any element as input/output. (RO11IT-349)
  • Improved the documentation of exposed REST APIs to provide better clarity on their support for cross-origin requests. (RPM-1217) Application Runtime Logic Execution
