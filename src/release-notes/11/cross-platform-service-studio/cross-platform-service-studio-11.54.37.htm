Cross-platform Service Studio 11.54.37

Released on Dec 11, 2023

New in Cross-platform Service Studio 11.54.37

  • Improved the perception of built-in functions by having different icons for client and server side actions. (RDEV-6096)
  • Added support for keyboard shortcuts to better manage dependencies namely remove unused references and update dependencies. (RDEV-6099)
  • Fixed an issue that caused publish to override latest platform version when a previous _undo_ on a automatic merge was performed. Users were unaware of publish override behavior. Now if such scenario is encountered the conflict dialog is always shown in the next publish. (RDPMBT-3741)

Bug Fixing

  • Fixed a crash that occurred during concurrent publishes. (RDEV-6091)
