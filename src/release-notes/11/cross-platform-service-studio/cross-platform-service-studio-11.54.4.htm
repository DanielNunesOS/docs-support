New in Cross-platform Service Studio 11.54.4

  • Code Mentor's intelligent quick search should now show results sooner for search queries starting with words commonly used in the suggestions' descriptions e.g. Call... (RAID-1902)

Bug Fixing

  • Fixed an issue that caused the Aggregates created using natural language to set the Max Records property to 1 when the Aggregate result was not a single record. (RAID-1595)
  • Fixed an issue that caused consumer modules to inappropriately refresh when clicking “Refresh all open Consumers” from the producer’s Module menu. This occurred when a producer module changed while a consumer deployed to a different environment was opened.  (RDOIBT-1247)
  • Fixed an issue when importing SOAP Web services containing document-type declarations (DTDs) (RPM-2866) Service Studio Logic
