Cross-platform Service Studio 11.54.8

Released on May 08, 2023

New in Cross-platform Service Studio 11.54.8

  • Improved loading time when creating a new Screen using Screen templates. (RDOIAT-1102)
  • Syntax errors in JavaScript nodes are now shown as warnings in the TrueChange panel. (In coherence with the interface Scripts) (RDOIBT-1327)

Bug Fixing

  • Fixed an issue that caused a hang when the user clicked multiple times on the compare and merge button. (RDOIAT-1045)
  • Fixed an issue causing the import of swagger files with references with multiple types to fail. (RDOIAT-1109)
  • Fixed an issue in Entity Edit Data that caused foreign key lookups to only load a maximum of 100 entities when a row was added. (RDOIBT-1336)
