Cross-platform Service Studio 11.53.43

Released on Mar 09, 2023

New in Cross-platform Service Studio 11.53.43

  • Improved Code Mentor logic suggestions to open or select the most likely property to be edited. Picking an Assign node now opens the Select Variable dialog, while picking an Execute Action node will select the first empty argument. Inspired by feedback from Carlos Olías. (RAID-1580)

Bug Fixing

  • Fixed an issue that caused the properties panel to be readonly when dropping a node in flow that would open a dialog. (RDOIAT-955)
  • Fixed an issue that created a duplicated aggregate when scaffolding a screen from an oracle external entity. (RDOIAT-961)
  • Fixed an issue that caused Service Studio to hang after submitting feedback of a crash while having some specific dialogs open. (RDOIAT-978)
  • Fixed an issue where the data type filter would not reset when clearing the search field. (RDOIAT-980)
  • Fixed an issue that prevented variables with name “Suggestions” to be selected in the properties' pane. (RDOIBT-1199)
  • Fixed an issue that caused the debugger to be blocked when using Chrome version 111.0.5563.64 or above. (RPM-3764) Service Studio Debugger
