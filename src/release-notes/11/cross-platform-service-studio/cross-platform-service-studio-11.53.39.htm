Cross-platform Service Studio 11.53.39

This is a preliminary version of the document.

New in Cross-platform Service Studio 11.53.39

  • It is now possible to see the historic of previously consumed REST Services. (RO11IT-263)

Bug Fixing

  • Fixed an issue that caused Publish not to happen on 1 click Publish. This occurred when Publishing with conflicts. Now, Publish will successfully work after Merging. (RDOIAT-922)
  • Fixed an issue that caused mappings to break when refreshing REST API. This occurred in a dropdown when the values of the dropdown would be mapped to a REST API.
    Now, the REST refreshes and the mappings are kept valid. (RMAC-10722)
  • Fixed an issue that caused new text to be created at the bottom of the screens when typing new text in placeholders. (RMAC-10738)
  • Now, output's default values are parsed correctly when consuming a REST service through a Swagger file. (RO11IT-80)
