Cross-platform Service Studio 11.54.9

Released on May 15, 2023

Bug Fixing

  • Fixed a bug that was hiding the “View Data” option for entities coming from external connections. (RDMST-1980)
  • Fixed an issue that could lead to wrong automatic merge decisions when publishing a version equal to an older one. (RDOIAT-1092)
  • Fixed an issue where the value of the Enabled field was automatically set to true after adding new entity attributes to the EditableTable, even if it was false. (RDOIBT-1339)
  • Fixed an issue that caused the What’s New dialog to be displayed without the arrows to move from item to item. (RICT-5195)
