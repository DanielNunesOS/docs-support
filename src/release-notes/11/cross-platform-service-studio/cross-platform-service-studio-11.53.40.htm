Cross-platform Service Studio 11.53.40

Released on Feb 20, 2023

New in Cross-platform Service Studio 11.53.40

  • Consuming REST services using multipart/form-data as request format. Inspired by Kilian Hekhuis's idea. (RO11IT-253)
  • Expose REST web services using the PATCH verb capability is now available. This was previously in technical preview but is now available for everyone. (RO11IT-27)

Bug Fixing

  • Fixed an issue causing the Icon widget to not be rendered in Traditional applications, inside Service Studio. (RDOIAT-913)
  • Fixed bottom pane flickering when clicking on 1 Click-Publish. (RDOIBT-1153)
