Cross-platform Service Studio 11.54.35

Released on Nov 27, 2023

New in Cross-platform Service Studio 11.54.35

  • Site properties can now be defined as a secret, which will ensure their value is stored and encrypted in the database. This requires LifeTime 11.20 or higher and Platform Server 11.25 or higher (RDOIBT-2019)

Bug Fixing

  • Fixed a crash that could happen while attempt to access a variable which type was a list of a list of records while debugging. (RDPMBT-3706)
