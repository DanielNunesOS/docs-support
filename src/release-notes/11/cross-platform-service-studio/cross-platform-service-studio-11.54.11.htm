New in Cross-platform Service Studio 11.54.11

  • Code Mentor logic suggestions now include Assigns with both variable and value more often, considering cases where their data types are not exactly the same. Bug Fixing

  • Fixed a bug that causes the Flow Editor to close after converting Actions to another Action kind. (RDEV-5658)
  • Fixed an issue that caused flow object text editions to be lost when navigating through the toolbar (i.e., 'Back' and 'Forward' arrows) while editing. (RDEV-5660)
  • Fixed an issue that caused development tabs to not render (i.e. TrueChange, Debugger, 1-Click Publish, and Search Results tabs). (RDEV-5676)
  • Fixed an issue that prevented the user to copy a Server Actions folder to Service Actions. (RDEV-5684)
  • Fixed an issue that caused a debugger error after changing the module's name. (RDEV-5721)
  • Fixed an issue that would cause rendering problems in Compare and Merge window. (RDEV-5723)
  • Fixed a crash when loading a traditional web module with public resources (RDMST-1972)
  • Fixed a crash when Replacing all usages of an element when it is currently used in the selected replacement value. (RDOIBT-1553)
  • Fixed an issue that could cause Service Studio to crash after the operating system deleted specific temporary files. (RDPMBT-3188)
