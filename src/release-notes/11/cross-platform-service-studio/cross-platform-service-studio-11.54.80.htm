Bug Fixing

  • Fixed an issue that caused tabs to becomed unresponsive while trying to close the application. (RDEV-7810)
  • Fixed an issue that caused timestamps in the 1-Click Publish messages panel to be displayed in the wrong format. (RDEV-7844)
  • Fixed an issue that caused an unexpected error when opening an OML file after loading a recent module. (RDEV-7856)
  • Fixed an issue that prevented the debugger from running for a module opened from the Recent Modules list after the module had been moved from one app to another. (RDEV-7862)
  • Fixed an issue that caused the "Send Details" button in the "Submit Feedback" window to sometimes be disabled when it shouldn't. (RDEV-7863)
