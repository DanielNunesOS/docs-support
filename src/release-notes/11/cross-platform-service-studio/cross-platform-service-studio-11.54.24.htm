Cross-platform Service Studio 11.54.24

Released on Aug 30, 2023

Bug Fixing

  • Fixed an issue that caused indirect references not to be imported after the merge conflict resolution step. (RDEV-5870)

  • Fixed an issue in login that caused error and warning dialogs to not be displayed after login from the browser using an Identity Service. (RDEV-5927)
  • Fixed an issue that caused a crash when attaching files in the Submit Feedback window. (RDMST-2185)
