Cross-platform Service Studio 11.53.42

This is a preliminary version of the document.

New in Cross-platform Service Studio 11.53.42

  • Improved the discoverability of the shortcuts associated to dialog buttons by adding the shortcut to the buttons tooltips. (RDOIAT-806)
  • It is now possible to use a keyboard shortcut (⌘+⌥+T on macOS and Ctrl+T on Windows) to test an advanced SQL query. (RDOIAT-818)

Bug Fixing

  • Fixed an issue that caused the input sentence for creating Aggregates using natural language to be completely replaced when a suggestion was selected from the dropdown. (RAID-1350)
  • Fixed an issue that was preventing the edition of Javascript files using unicode chars. (RDOIBT-1164)
