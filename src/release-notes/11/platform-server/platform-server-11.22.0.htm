Platform Server 11.22.0

Released on Jul 03, 2023

New in Platform Server 11.22.0

  • Upgraded ITfoxtec.Identity.Saml2 library to version 4.8.6. (R11PIT-1209)

Bug Fixing

  • Fixed an issue that prevented the correct functionality of effective user provider feature when using Redis. (R11BRT-435)
  • Fixed issues when accessing the "Distribute Mobile App" tab in Service Studio, that were caused by having external authentication enabled on the platform. (RPM-2696) (RPM-2697) (RPM-2293) Application Runtime System Components
  • It's now possible to override the default logic used to determine how connections are authenticated against LDAP servers for End-Users and IT users, by configuring a Site Property in Users or LDAPAuthProvider modules, respectively. This can be used to allow integration with servers using particular setups. (RPM-2987) Application Runtime Authentication and Authorization
  • Fixed compilation on Service Studio by disallowing Service Actions to have inputs or outputs Default Values pointing to a Static Record Id. (RPM-332) Publish Operation Compilation

The calculation of dependencies was selecting the wrong version during the first publication and, due to this, the referenced site properties were not being published. (RPM-3955) Publish Operation Compilation
  • Updated Open-Source projects Licenses and Copyrights used in the Platform (RPM-3978) Mobile Apps Builder
