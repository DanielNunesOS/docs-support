Platform Server 11.16.0

This is a preliminary version of the document.

New in Platform Server 11.16.0

  • Improved the performance of parsing SQL nodes query text during applications runtime when using expand inline inputs (R11PBT-588)
  • Improved the performance of solution publishing. These improvements will be gradually rolled out to OutSystems customers and won’t be enabled by default when upgrading to this version. (R11PBT-613)
  • Disabled Oracle "Out Of Band Breaks" feature in all connections to improve the compatibility with typical network and firewall installations. Having the feature enabled could lead for query timeouts not being respected. (R11PBT-620)
  • Upgraded the Oracle Managed Driver to version 19.14.0. For more information on related fixes, see the Oracle release notes. (R11PBT-667)
  • Updated NLog library bundled with the platform from version 4.7.8 to version 4.7.14. (R11PBT-682)
  • Improved the performance of solution publishing. These improvements will be gradually rolled out to OutSystems customers and won’t be enabled by default when upgrading to this version. (R11PBT-701)
  • Fixed an issue that could cause threads to hang in Oracle environments. (RPM-1995)

Bug Fixing

  • Fixed an issue that caused IT users to lose permissions over the applications of an environment. (R11CT-1106)
  • Fixed compilation error when consuming SOAP Web Services containing nested elements with the same name. (R11DT-963)
  • Fixed compilation crash when referencing an incompatible Service module (R11PBT-590)
  • Fixed an issue that prevented CSP headers from being sent in Service Center responses for some platform installation scenarios. (R11PBT-725)
  • Fixed an issue that could cause protocol errors in requests to Service Center when requests are handled using the HTTP/2 protocol. (R11PBT-726)
  • Symbolik link creation now fallback to directory copy when user lacks permissions (R11PIT-639)
  • Fixed an issue that could cause the Scheduler Service to become unhealthy in Oracle environments (RPM-1133)
  • Http Server information is now removed from response header. (RPM-1461)
  • Fixed an issue that caused email sending to be aborted when Office 365 returns error 550 (Recipient address rejected). (RPM-1514)
  • Fixed an issue that could cause IIS requests to hang in Oracle environments (RPM-1573)
  • All Reactive and Mobile applications running inside an iframe now behave as running on the browser even if the parent is a native app. (RPM-1711)
  • When executed in a Lifetime environment, Platform installer will now check if version being installed matches the expected version for the Lifetime version installed. (RPM-1932)
  • Fixed compilation error when consuming SOAP Web Services containing nested elements with the same name. (RPM-2059)
  • Fixed an issue that caused a compilation error when handling references to producer's non-basic types (e.g structures) which no longer exist. (RPM-2071)
  • Fixed a bug in the Dropdown Widget when validating with an empty value defined. (RPM-2123)
  • Fixed an issue when the EXIF finds an unexpected EXIF property. (RPM-2138)
  • Fixed an issue in Service Center that caused the display of wrong results when filtering the modules list by "errors and warnings" state. (RPM-2139)
  • Fixed an issue that caused a warning, stating that the User Provider from that module was outdated, to persist in Service Center even when the error was not happening. (RPM-2146)
  • Now the "Timer_NativeBuildTracker" in Service Center has a higher timeout value than the default value returned from MABS. This value is configurable through the new site property "NativeBuildTimeoutMargin". (RPM-2147)
  • Fixed an issue in Service Center that caused the display of wrong results when filtering the modules list by "errors and warnings" state. (RPM-2155)
  • Fixed Oracle semaphore related errors in high concurrency scenarios. This issue is fixed by the Oracle Driver upgrade done in R11PBT-667. (RPM-2178)
  • Granting a security level to a user on a Database Connection no longer adds that level to all available connections. (RPM-2277)
  • Fixed scenarios in which suspended process instances got resumed without any change to the process definition (RPM-2290)
  • Fixed a concurrency issue with the DatabaseConnection_SetConnectionStringForSession method of the PlatformRuntime_API that could lead to the incorrect connections to be used. (RPM-2340)
  • Fixed a bug in the Upload Widget when selecting the same image twice in Chrome. (RPM-446)
  • Added support for reading the External ID retrieved during SAML Authentication inside an application. (RPM-671)
  • Fix a wrong behavior when using the browser back button or Previous Screen event to navigate back to a screen with Widgets that use data from Client/Local variables. The Widgets was not re-rendered and showed outdated values. (RPM-673)
  • Improved brute force protection messaging when authenticating against Server.Identity. (RPM-836)
  • Fixed an issue that caused all modules to show a warning stating that they need to be recompiled when adding a new front-end in a farm scenario. (RPM-951)
  • Fixed a user privilege vulnerability in Service Center where AD groups were not taking precedence over native users. CVSSv3.0 score 4.2 (Medium). (RPM-2056)
  • Fixed an issue that caused sensitive values to be captured by the App Feedback's feedback on Reactive applications. CVSSv3.1 score 4.2 (Medium). (RPM-2091)
  • Fixed a security issue that was causing Configuration Tool to log credentials in its log file. CVSSv3.1 score 5.5 (Medium). (RPM-2136)
