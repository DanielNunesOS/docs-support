Platform Server 11.16.1

This is a preliminary version of the document.

Bug Fixing

  • Fixed the issue on DropdownMenu widget from RichWidgets that forces the end user to click twice when collapsing the dropdown when used the SideMenu layout (Applies only to Traditional Web apps) and Fixed the issue that never closes the menu on a body click. (ROU-3331)
  • Fixed an error when setting IT users permissions and there are more than 1000 application permissions. This issue applied only to environments using an Oracle database. (RPM-1323)
  • Fixed a validation issue in the Dropdown widget for Reactive Web and Mobile apps when the "Empty Text" property is set. (RPM-2123)
  • Fixed an issue that prevented attributes from being correctly imported into an entity using Integration Studio when importing an entity from a View using an external PostgreSQL database connection (RPM-2388)
  • Now, the DropdownMenu closes on a body click. (RPM-2409)
  • Improved telemetry query in Service Center (RPM-2539)
  • Removed redundant sync environment from solution publish. (RPM-819)

More details

RPM-1323
Fixed an error when setting IT users permissions and there are more than 1000 application permissions. This issue applied only to environments using an Oracle database.
Application Lifecycle LifeTime

Fix Details:
When registering a new Oracle environment in the infrastructure through the LifeTime console, the environment synchronization generated several errors in the advanced query RemoveDeprecatedPermissions, which is executed by an internal Platform Server API. This problem only occurred for Oracle environments having more than 1000 application permissions.

RPM-2123
Fixed a validation issue in the Dropdown widget for Reactive Web and Mobile apps when the "Empty Text" property is set.
Application Runtime Interface

Fix Details:
This issue occurred in Reactive Web Apps and Mobile Apps having a Dropdown widget with the "Empty Text" property set. When a value was selected in the Dropdown, after clearing that value by setting the bound variable value to its default, the Widget would incorrectly pass the required validation although no value was selected.

RPM-2388
Fixed an issue that prevented attributes from being correctly imported into an entity using Integration Studio when importing an entity from a View using an external PostgreSQL database connection
Integration Studio Import Wizards

Fix Details:
*Symptoms* Incorrect attributes imported when consuming certain Views in Integration Studio using the OutSystems PostgreSQL External Database Connector.

RPM-819
Removed redundant sync environment from solution publish.
Application Lifecycle LifeTime

Fix Details:
*Symptoms* Deadlock errors during synchronization in LifeTime. *How to reproduce* This problem is already minimized in recent versions of the platform (LT 11.8.0+) that have improvements over the Synchronization process, compared to older versions.
