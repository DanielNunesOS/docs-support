Platform Server 11.23.0

Released on Aug 21, 2023

New in Platform Server 11.23.0

  • Added support for REDIS 7.0.11 (or higher, within the 7.0 line) as the session database for on-premise customers. Support for REDIS is no longer a technical preview. (R11BRT-39)
  • It is no longer possible to create or use Deployment Zones with hosting technology set to containers. If you have any applications assigned to a Deployment Zone with hosting technology set to containers, you will need to manually move them to a Deployment Zone with hosting technology set to Classical Virtual Machines. (R11BRT-410)
  • Now, the integrations tab of Module details in ServiceCenter has the lists of integrations sorted alphabetically. (R11CT-2168)
  • Added a new audit message when an IT user changes the administrator password for a user provider module in ServiceCenter. (R11CT-2283)
  • Updated Open-Source projects Licenses and Copyrights used in the Service Studio. (R11DT-1852)
  • Improved free form search on ServiceCenter Monitoring pages. (R11OT-593)

Bug Fixing

  • Fixed an issue that displayed an error when a Traditional or Reactive application sends emails. When sending emails from an application with a Site Rule configured, the error message displayed was: "Error creating Email. The remote server returned an error: (404) Not Found". The fix is disabled by default and can be enabled by changing a system parameter. (RPM-2156) Application Runtime Interface
  • Improved performance of the CleanLoginAttempts timer from the Users module. (RPM-2619) Application Lifecycle Users
  • Advanced queries with multiple lines no longer result in a compilation error (RPM-2938) Publish Operation Compilation
  • Fixed an issue that caused the request URL to a custom path to be incorrect when opening a Screen/WebBlock that contained the pattern in their name ("__"). This occurred in Reactive and Mobile Apps. (RPM-3033) Application Runtime Interface
  • Fixed an issue that caused duplication of HSTS headers. (RPM-3519) Application Runtime Interface
  • Fixed an issue that caused an error when staging a producer of an ISV-protected module in LifeTime. (RPM-3820) Application Lifecycle LifeTime
  • Fixed performance degradation experienced when navigating through Service Center pages, that could happen after a Modules Preparation operation ended with errors. (RPM-3923) Infrastructure Management Platform Configuration
  • Now, a feedback message will appear in ServiceCenter when trying to generate a mobile application that doesn't have a home module defined. (RPM-3937) Application Lifecycle Service Center
  • Fixed an issue that caused the request URL to be incorrect (without the hash content) when opening a Module/Screen/WebBlock that contained the pattern in their name ("__"). This occurred in Reactive and Mobile Apps. (RPM-4033) Application Runtime Logic Execution
  • Fixed an issue that caused successful publications through OSPTool to incorrectly exit with code -4. (RPM-4052) Infrastructure Management OSPTool
  • Popover Widget is now focusable and clickable by keyboard, enhancing usability for keyboard-only users. (RPM-4080) Application Runtime Interface
  • Fixed an issue where the aria-describedby parameter was being added to an input widget, even when there wasn't any validation error message. (RPM-4088) Application Runtime Interface
  • Fixed an error in ServiceCenter that would happen when running a background process that removes deprecated IT user permissions related to applications. This would happen when running on Oracle and an IT user would have access to more than one thousand applications. (RPM-4113) Application Lifecycle LifeTime
  • Fixed an issue that caused an internal error when publishing license protected modules in trial, demo and personal environments. (RPM-4168) Publish Operation Deployment stage
  • Fixed a compilation issue when consuming a SOAP web service with xsd:choices with xsd:restrictions. (RPM-4209) Application Runtime Logic Execution
  • Fixed an issue that caused Invalid Permissions screen to appear in ServiceCenter when clicking "Back to module" on the SiteProperty and Timer edit screens. (RPM-645) Application Lifecycle LifeTime
  • Fixed a security vulnerability that could allow environment information disclosure. CVSSv3.1 score 6.5 (Medium) (RPM-4068) Application Lifecycle Service Center
  • Fixed a Server-Side Request Forgery vulnerability. CVSSv3.1 score 5.9 (Medium). (RPM-4126) Application Lifecycle Service Center
