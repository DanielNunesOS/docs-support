ODC Studio 1.2.3

Released on Sep 25, 2023

Bug Fixing

  • Fixed a bug where searchable dropdowns would show no elements when filtering them with an empty search term. (RDEV-5926)
  • Fixed a bug that caused Copy Paste via shortcut keys to stop working on certain scenarios. (RDEV-5938)
  • Fixed an issue that caused the Studio to hang when a debug session stopped on a breakpoint while the Styles pane was visible. (RDEV-5941)
  • Fixed a crash when using parameters with the same name but in different actions in the execute node. (RDMST-2259)
  • * Fixed an issue where clicking into and invalid item in the find results table could lead to a crash. * Now, activating a find results tab will make it to be automatically refreshed, non-existing items will be removed from the table. (RDPMBT-3595)
  • Fixed an issue that caused a crash when a user clicks on a block shown in a screen while comparing the versions in merge editor. (RDPMBT-3597)
  • Fixed a crash when opening the context menu and a request failed to open a screen in the browser. (RDPMBT-3598)
  • Fixed “Get palette from image” action in theme editor. (RDPMBT-3599)
