ODC Studio 1.4.16

This is a preliminary version of the document.

New in ODC Studio 1.4.16

  • Fixed security vulnerabilities with Low Severity. (RDEV-7316)
  • Referenced server actions consumed in screens accessible by unauthenticated users will generate a security warning. (RDEV-7376)

Bug Fixing

  • Fixed an issue that would cause ODC Studio to be unresponsive when switching tabs. (RDEV-7345)
  • Fixed an issue that caused the file picker dialogs of "Save Resource as.." and "Change Resource.." to filter by extension name incorrectly. (RDEV-7396)
  • Fixed an issue that could cause an unexpected error when dragging a widget while an Expression Editor dialog was open for an argument of the widget being dragged. (RDEV-7411)
  • Fixed an issue that would cause an error when hiding a tooltip. (RDEV-7417)
  • Fixed an issue that could cause a crash while debugging multiple applications at the same time. (RDEV-7419)
  • Fixed Interface tab's name. (RDEV-7424)
