ODC Studio 1.4.5

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue that could cause an unexpected error when updating a Library while consuming its structures. (RDEV-6489)
  • Fixed an issue that caused ODC Studio to hang when opening or publishing an app or library. (RDEV-6492)
  • Fixed an issue where the publishing process of some apps was aborted due to a concurrent publishing error when a merge had occurred prior to the app being uploaded. (RDEV-6526)
