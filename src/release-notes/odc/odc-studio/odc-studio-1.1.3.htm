ODC Studio 1.1.3

Released on Jun 05, 2023

New in ODC Studio 1.1.3

  • Code Mentor logic suggestions now include Assigns with both variable and value more often, considering cases where their data types are not exactly the same. Common examples include assigning a Product Record List to a Product List variable, or an Integer to a Decimal variable. Inspired by feedback from Carlos Olías. (RAID-1951)

Bug Fixing

  • Fixed an issue that caused Code Mentor logic suggestions to not allow building simple CRUD wrapper actions completely. Now the suggested Assign nodes include the previous node’s output as value. Reported by Vincent Koning. (RAID-666)
  • Fixed an issue that caused the debug session to stop after the application being debugged is published by another developer. (RDEV-5580)
  • Fixed an issue that caused flow object text editions to be lost when navigating through the toolbar (i.e., 'Back' and 'Forward' arrows) while editing. (RDEV-5660)
  • Fixed an issue that caused development tabs to not render (i.e. TrueChange, Debugger, 1-Click Publish, and Search Results tabs). (RDEV-5676)
  • Fixed an issue that prevented the user to copy a Server Actions folder to Service Actions. (RDEV-5684)
  • Fixed an issue that would cause rendering problems in Compare and Merge window. (RDEV-5723)
  • Fixed an issue where background dialogs could be closed dispite having a blocking dialog in front. (RDOIBT-1557)
  • Fixed an issue that prevented the interaction in the True Change panel after hiding a warning using a keyboard shortcut. (RDOIBT-1566)
  • Fixed an hang after merge operation with complex modules. (RDPMBT-3236)
