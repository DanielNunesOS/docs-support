ODC Studio 1.1.1

Released on May 22, 2023

New in ODC Studio 1.1.1

  • Fixed a security vulnerability, CVE-2022-34716. (RICT-5172)

Bug Fixing

  • Fixed an issue where we didn't accept commas consuming REST APIs (RDEV-5678)
  • Fixed a bug that was hiding the “View Data” option for entities coming from external connections. (RDMST-1980)
  • Fixed an issue in Entity Edit Data that caused foreign key lookups to only load a maximum of 100 entities when a row was added. (RDOIBT-1336)
  • Fixed an issue in Entity Edit Data that caused foreign key lookups to only load a maximum of 100 entities when a row was added. (RDPMBT-3147)
  • Fixed an issue that caused the What’s New dialog to be displayed without the arrows to move from item to item. (RICT-5195)
