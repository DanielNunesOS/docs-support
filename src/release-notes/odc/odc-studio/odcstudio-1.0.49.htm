ODCStudio 1.0.49

Released on Apr 17, 2023

New in ODCStudio 1.0.49

  • Suggestions for inline records' fields will now use name matching to narrow down available options. (RAID-1899)
  • You can now export your organization list from the “Manage organizations” dialog. (RDEV-5524)
  • It is now possible to use keyboard shortcuts on Find and Replace buttons. (RDEV-5547)
  • It is now possible to use keyboard shortcuts on Give us Feedback dialog buttons. (RDEV-5549)
  • It is now possible to integrate entities from external systems. Developers can integrate with SQL Server and Oracle faster than ever. (RDPMBT-3076)

Bug Fixing

  • Fixed an issue that caused debugger variables to appear out of order. (RDEV-5543)
  • Fixed an issue in the debugger that prevented users from inspecting variables that contain nested lists. (RDEV-5548)
  • Fixed an issue that could cause hangs when debugging applications with the "Break On All Exceptions" enabled. (RDOIAT-1047)
