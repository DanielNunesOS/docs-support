ODCStudio 1.0.43

This is a preliminary version of the document.

Bug Fixing

  • Fixed an issue that caused the input sentence for creating Aggregates using natural language to be completely replaced when a suggestion was selected from the dropdown. (RAID-1350)
  • Fixed an issue that prevented consumers of a library from refreshing dependencies automatically when the library is published and the consumer is opened locally. (RDEV-5436)
  • Fixed an issue that prevented users from dropping widgets next to other widgets that were filling all the available space in a container. (RDPMBT-2383)
