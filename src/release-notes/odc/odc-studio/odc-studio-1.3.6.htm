ODC Studio 1.3.6

Released on Dec 04, 2023

New in ODC Studio 1.3.6

  • On View/Edit Data and Aggregates screens, the headers containing the name of the attributes, are now suffixed with (UTC) in the case of attributes with the Date, DateTime and Time types. (RDOIBT-1901)

Bug Fixing

  • Fix an issue that adds Users module when REST authentication is changed to basic. Now it should not get added on changing REST authentication to basic. (RDPMBT-3755)
