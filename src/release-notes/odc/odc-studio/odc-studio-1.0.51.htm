ODC Studio 1.0.51

Released on May 01, 2023

Bug Fixing

  • Fixed an issue that causes a crash when application templates fail to load while creating a new application. (RDEV-5632)
  • Fixed a rare crash scenario that could happen while trying to internally detect hangs during progress operations. (RDEV-5641)
  • Fixed an issue that caused an error when loading a Swagger for REST consumption when having a schema reference with multiple types (oneOf). (RICT-5137)
  • Fixed an issue in Windows that caused lost of focus when opening AI suggestions menu. (RICT-5139)
  • Fixed an issue in the Windows installer that caused a freeze when clicking in the installer dialog. (RICT-5167)
