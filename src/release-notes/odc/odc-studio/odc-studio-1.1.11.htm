ODC Studio 1.1.11

Released on Aug 07, 2023

New in ODC Studio 1.1.11

  • Code Mentor now provides extra logic suggestions in client-side action flows to accelerate common scenarios. For example, searching for “Error” or “Success” provides specialized Feedback Message suggestions. (RAID-1904)

Bug Fixing

  • Fixed issue that caused Code Mentor logic suggestions to open a picker in scenarios where it was not necessary. For example, picking a “Raise Invalid Login exception” suggestion would open the Exception picker, even though the exception was already filled in. (RAID-2003)
  • _Fixed an issue that caused copy-paste of tree elements within app layer tabs not to work when using keyboard shortcuts._ (RDEV-5828)
  • Fixed an issue that caused the focus to be lost when deleting warnings using keyboard shortcuts. (RDOIBT-1694)
