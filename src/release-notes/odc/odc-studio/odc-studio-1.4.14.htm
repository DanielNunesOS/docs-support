ODC Studio 1.4.14

Released on Jun 18, 2024

New in ODC Studio 1.4.14

  • The searchable dropdown in the 'Add Public Elements' dialog was redesigned to improve usability by integrating the search bar directly within the option menu. (RDEV-5951)
  • You can now see the executed SQL query of an Aggregate in the Properties pane. (RDEV-6980)

Bug Fixing

  • It is now possible to directly drag a Service Action to a flow of another application. (RDEV-6672)
  • Fixed an issue that would cause ODC Studio to be unresponsive during compare and merge operations. (RDEV-6982)
  • Fixed a scenario where undoing changes done in the language editor while editing the style classes property of widgets would open the CSS editor instead of the language editor (RDEV-7287)
