ODC Studio 1.4.6

This is a preliminary version of the document.

New in ODC Studio 1.4.6

  • Improved the icon of the ListAppendAll built-in function. (RDEV-6509)

Bug Fixing

  • Fixed an issue that causes an indefinite loading when opening an app. (RDEV-6476)
  • Fixed an issue that caused an unexpected error in ODC Studio when pasting a global event into an app where the event's input parameter type wasn't yet available in the app. (RDEV-6553)
  • Fixed an issue that caused a communication error after the login dialog was closed without logging in. This occurred when the refresh token expired while opening "Add public elements" dialog. (RDEV-6568)
