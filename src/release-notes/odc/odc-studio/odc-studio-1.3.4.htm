ODC Studio 1.3.4

Released on Nov 20, 2023

New in ODC Studio 1.3.4

  • The "Open other revision" dialog can now list more than just the last 100 revisions. (RDEV-5876)
  • Studio now allows to debug on Edge browser. (RDEV-6098)
  • The new dialog to edit application/library properties can now be accessed by clicking the application/library name in the top menu bar or via the hamburger menu option Edit app properties. (RDPMBT-3754)

Bug Fixing

  • Fixed an issue that would cause tutorials not to work correctly. (RDEV-6046)
