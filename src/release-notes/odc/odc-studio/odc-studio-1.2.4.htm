ODC Studio 1.2.4

Released on Sep 27, 2023

New in ODC Studio 1.2.4

  • Now, apps and libraries installed from forge will only be allowed to be open and modified by its owner. Their behavior will be similar to supported or protected apps, and a dialog asking about cloning the application if further customizations are needed. (RDPMBT-3608)

Bug Fixing

  • Fixed a bug that caused a communication error when clicking "Compare and merge with another revision or file" for an app or library opened from the file system. (RDEV-5949)
  • Fixed an issue that redirected the user to an O11 help page instead of ODC in the help dialog. (RDEV-5954)
  • Fixed a crash on IDE startup when clicking to connect while automatic login is in progress. (RDOIBT-1836)
  • Fixed an issue that caused a wrong title of a screen and text of a button when creating screens using scaffolding. (RDPMBT-3600)
