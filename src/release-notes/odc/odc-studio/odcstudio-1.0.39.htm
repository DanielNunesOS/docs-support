ODCStudio 1.0.39

Released on Feb 02, 2023

New in ODCStudio 1.0.39

  • ODC Studio now allows developers to check for dependency updates on demand. (RDPMBT-2589)
  • ODC Studio now allows developers to select all available public elements of a source app or library when adding public elements to their app through the Add Public Elements dialog. (RDPMBT-2605)
