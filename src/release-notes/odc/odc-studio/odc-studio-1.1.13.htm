ODC Studio 1.1.13

Released on Aug 21, 2023

Bug Fixing

  • Fixed an issue where the _Help with “Aggregates”_ option was not opening the correct help page. (RDOIBT-1671)
  • Fixed an issue that caused the replace data of entity attributes into web-screens to fail when replacing binary data type attributes. (RDOIBT-1728)
  • Fixed an issue where foreign key dropdowns were not available the first time a foreign key attribute was added. (RDOIBT-1730)
  • Fixed an issue that caused a crash when opening the search dialog (RDOIBT-1794)
