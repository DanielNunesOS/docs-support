ODCStudio 1.0.47

Released on Apr 03, 2023

New in ODCStudio 1.0.47

  • You can now access a useful documentation link to help choosing between the creation of the three different types of App templates on ODC Studio's "Create" dialog. (RDEV-5231)
  • The Debugger pane now does not render a "Entry app" dropdown if the currently open tab corresponds to an application. The same applies to the “Select entry app…” Debugger menu item. (RDEV-5423)

Bug Fixing

  • Fixed an issue that caused loss of focus when moving an attribute down using the keyboard shortcut Ctrl+Down on Windows or ⌘+⌥+Down on MacOS. (RDEV-5486)
  • Fixed an issue in login that caused the wrong error message to be displayed when the organization is not found. (RDEV-5498)
  • Fixed an issue that caused some libraries to be considered templates while opening. (RDEV-5512)
  • Fixed an issue that caused the ODC Studio to show a wrong image when facing an empty aggregate. (RDEV-5513)
  • Fixed an issue that prevented multiple login dialogs to be displayed across different ODC Studio instances. (RDEV-5523)
