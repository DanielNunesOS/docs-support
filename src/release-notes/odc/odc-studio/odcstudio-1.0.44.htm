ODCStudio 1.0.44

This is a preliminary version of the document.

New in ODCStudio 1.0.44

  • Improved Code Mentor logic suggestions to open or select the most likely property to be edited. Picking an Assign node now opens the Select Variable dialog, while picking an Execute Action node will select the first empty argument. Inspired by feedback from Carlos Olías. (RAID-1580)
  • It is now possible to use a keyboard shortcut (⌘+⌥+T on macOS and Ctrl+T on Windows) to test an advanced SQL query. (RDEV-5452)

Bug Fixing

  • Fixed an issue that was preventing the edition of Javascript files using unicode chars. (RDOIBT-1164)
