ODC Studio 1.4.10

Released on May 20, 2024

New in ODC Studio 1.4.10

  • Double-clicking Events now redirects to their Handler action. (RDEV-6499)
  • It is now possible to search for specific elements in the Scope Tree while using expression editors, enhancing navigation and efficiency. (RDEV-6607)
  • Added support for macOS Ventura and macOS Sonoma. (RDEV-6958)

Bug Fixing

  • Fixed an issue that would prevent users to login into ODC Studio. (RDEV-6647)
  • Fixed an issue that would cause ODC Studio to hang after publishing. (RDEV-6661)
