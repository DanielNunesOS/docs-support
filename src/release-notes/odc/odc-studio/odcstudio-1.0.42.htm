ODCStudio 1.0.42

Released on Feb 24, 2023

New in ODCStudio 1.0.42

  • Native Menus from MacOS now include an icon as a visual aid to help distinguish between commands. (RDPMBT-2672)
  • Native Menus from MacOS now include an icon as a visual aid to help distinguish between commands. (RTAFB-5870)

Bug Fixing

  • Fixed an issue that caused the application logo not to be correctly rendered in the Edit Application Details dialog. This occurred when the logo was updated. Now, after re-opening the Edit Application Details, the logo is rendered correctly. (RDEV-5403)
  • Fixed an issue that caused the toggle landscape node to be enabled for desktop view. (RDEV-5428)
  • Fixed an issue that caused the add public elements table header to have a wrong color. (RDEV-5433)
  • Fixed Ctrl/CMD+N shortcut not working properly to create new elements while editing tree node. (RDPMBT-2440)
