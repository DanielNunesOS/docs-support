ODC Studio 1.4.2

Released on Mar 26, 2024

New in ODC Studio 1.4.2

  • The search for logic elements in Code Mentor (Intelligent Quick Search) is now expanded by default, making it easier for you to continue building a flow even when the logic suggestions were not what you were looking for. (RAID-2351)
  • Users can now open their Timers in the Portal directly from Service Studio. Inspired by Kilian Hekhuis ’ idea. (RDEV-6188)
