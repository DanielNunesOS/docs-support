ODC Studio 1.4.3

This is a preliminary version of the document.

New in ODC Studio 1.4.3

  • Improved handler creation for events by automatically assigning the events input parameters to the newly created server action. (RDEV-6097)
  • Fixed an issue in Test SQL that caused 'publish required' error message to be incorrectly displayed. (RDEV-6429)

Bug Fixing

  • Fixed an issue that causes ODC studio to become unresponsive when a user updates library within few seconds of completing publish. (RDEV-6446)
  • Fixed an issue that causes an indefinite loading when opening an app. (RDEV-6476)
