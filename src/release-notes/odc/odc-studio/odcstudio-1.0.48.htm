ODCStudio 1.0.48

Released on Apr 10, 2023

New in ODCStudio 1.0.48

  • Code Mentor's intelligent quick search should now show results sooner for search queries starting with words commonly used in the suggestions' descriptions e.g. Call... (RAID-1902)
  • Now, when opening the "add public elements" dialog from a Reference app or library, the Source filter will be automatically set to that app or library. (RDEV-5474)

Bug Fixing

  • Fixed an issue that caused the Aggregates created using natural language to set the Max Records property to 1 when the Aggregate result was not a single record. (RAID-1595)
  • Fixed an issue in SQL editor that caused the editor to close after pressing ESC to cancel a tree node's name edition. (RDEV-5533)
  • Fixes issue that causes a hang when opening dialogs. (RDEV-5535)
  • Fixed an issue in the New App Dialog that caused ODC Studio to crash when loading application templates. (RDEV-5537)
  • Fixed an issue when showing submit feedback sometimes results in a hang. (RDOIAT-1000)
  • Improves the performance when performing searches inside the module. (RDOIAT-1003)
  • Fixed an issue that could occasionally cause a hang due to race conditions when opening dialogs. (RDOIAT-1004)
