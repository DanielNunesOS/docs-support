ODC Studio 1.4.32

Released on Nov 18, 2024

New in ODC Studio 1.4.32

  • Improve organization of context menu options for importing connections. (RDEV-7928)

Bug Fixing

  • Fixed an issue that caused an unexpected error when refreshing a REST API that had multiple actions with the same URL Path value. (RDEV-7927)
  • Fixed an issue that could cause an error when refresh dependencies operations are cancelled. (RDEV-7948)
  • Fixed an issue that caused an unexpected error when pressing ESC while a tooltip is open in a blocking dialog. (RDEV-7971)
  • Fixed an issue that caused dates to be displayed in the wrong format. (RDEV-7972)
