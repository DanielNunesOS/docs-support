ODCStudio 1.0.50

Released on Apr 24, 2023

New in ODCStudio 1.0.50

  • It is now possible to detach and reattach module tabs into existing Service Studio windows. (RDPMBT-2971)

Bug Fixing

  • Fixed issue that caused attribute mappings to not be automatically filled in some scenarios where the attribute names and data types completely match. (RAID-1900)
  • Fixed an issue that caused consumer apps to inappropriately refresh when a producer library was published. This occurred when a producer library changed while a consumer app deployed on a different environment was open.  (RDEV-5564)
  • Fixed an issue that caused a failure connecting to Chrome when starting Debugger when another Chrome instance was left open from a previous debug session. (RDEV-5602)
  • Fixed a bug that caused a crash while expanding some structure variables in use during a debug session. (RDEV-5606)
