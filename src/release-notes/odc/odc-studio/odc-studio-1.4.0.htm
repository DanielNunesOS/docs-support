ODC Studio 1.4.0

Released on Mar 13, 2024

New in ODC Studio 1.4.0

  • It is now possible to publish and consume events between ODC applications. (RDEV-6423)

Bug Fixing

  • Fixed an issue that redirected the user to an O11 help page instead of ODC in the help dialog. (RDEV-5954)
  • Fixed an issue that prevented end user from selecting empty popup after expanding it and dropping widgets in it. (RDEV-6326)
  • Fixed an issue where selecting the "Help" option, when right-clicking a 1-Click Publish message, failed to open the "Help" dialog. (RDEV-6377)
  • Fixed an issue that caused ODC Studio to crash when saving an OML while the Edit app/library properties dialog was open. (RDEV-6380)
  • Fixed an issue that could cause ODC Studio to crash when closing an app/library while it was publishing. (RDEV-6407)
  • Fixed an issue that could cause ODC Studio to enter a publish loop when publishing an app/library with incompatible references. (RDEV-6412)
  • Fixed a hang that occasionally occurred while starting or refreshing a Try Library session. (RDEV-6413)
  • Fixed an issue that was preventing Icons to be shown when dragging the Icons Widget into a Block or Screen. (RDMST-2694)
