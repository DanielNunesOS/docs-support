ODC Studio 1.4.29

Released on Oct 28, 2024

Bug Fixing

  • Fixed an issue that caused tabs to becomed unresponsive while trying to close the application. (RDEV-7810)
  • Fixed an issue that caused the minimize button to stop working on macOS Sequoia. (RDEV-7822)
  • Fixed an issue that caused timestamps in the 1-Click Publish messages panel to be displayed in the wrong format. (RDEV-7844)
  • Fixed an issue in extensibility configuration for mobile apps where auto-complete was suggesting invalid resources. (RDMDX-3)
  • Fixed an issue where libraries would suggest resources in the extensibility configuration editor. (RDMDX-5)
