ODC Studio 1.2.0

Released on Sep 04, 2023

New in ODC Studio 1.2.0

  • It is now possible to edit entity data directly in ODC Studio. (RDOIBT-1822)

Bug Fixing

  • Fixed an issue that caused indirect references not to be imported after the merge conflict resolution step. (RDEV-5870)
  • Fixed an issue that caused a crash on any save action when opening an editor. (RDEV-5905)
  • Fixed an issue that caused a crash when attaching files in the Submit Feedback window. (RDMST-2185)
