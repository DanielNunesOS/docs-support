OutSystems Developer Cloud 2023-09-18

Released on Sep 18, 2023

New

  • When seeing the users associated to an organization role, we now also show the user’s email to better distinguish between users with the same name. (RDPP-1187)
  • Updated the invite user screen to show only 3 end-user groups at a time, to improve the screen’s readability. (RDPP-1295)
  • When inviting new users, the end-user group cards now have a link on the group name to see the group’s details. This allows users to gain more context about each group while assigning them. (RDPP-1296)
  • Updated the terminology used throughout ODC Portal when referring to a workforce user’s access privileges from “Member access” to “Organization Access”, to make it more clear what it refers to. (RDPP-1375)

Bug Fixing

  • Fixed an issue in the user’s profile page that could cause the filters for assigned end-user roles to disappear. (RDPP-1254)
  • Fixed an issue that could surface when searching for users within the scope of a specific end-user group. (RDPP-1389)
  • Fixed an issue in the end-user access tab of the user’s profile page on ODC Portal, that could cause a Forbidden error message to be shown while using the apps filter. (RDPP-1413)
  • Fixed an issue when managing the group mappings for an Identity Provider on ODC Portal, where the filters would not work correctly in case there were no results. (RDPP-1422)
  • Fixed an issue where the user could see one or more error messages when accessing the Identity Providers list on ODC Portal, if he did not have permission to manage identity providers. (RDPP-1434)
