OutSystems Developer Cloud 2024-04-16

Released on Apr 16, 2024

New

  • Monthly Security Update (March 2024): Updated application container base image, including security fixes. Check Microsoft .NET 8.0.3 Release for more details: https://github.com/dotnet/core/blob/main/release-notes/8.0/8.0.3/8.0.3.md (RRCT-5689)

Bug Fixing

  • Fix an issue that caused users' email addresses to not be automatically marked as verified when users entered the ODC Portal through an external identity provider, which was preventing users from logging into the ODC Portal. Now, the email is automatically marked as verified, simplifying the login process for users accessing the ODC Portal via an external identity provider, ensuring a smoother and (RDPIDT-2103)
