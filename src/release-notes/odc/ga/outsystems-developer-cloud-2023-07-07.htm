OutSystems Developer Cloud 2023-07-07

Released on Jul 07, 2023

New

  • This release introduces a new failure reason, "TooManyFailedAttempts," to the Client action "ChangePassword." In the event of a failed change password flow due to excessive unsuccessful attempts, users will receive a specific message: "Too many failed attempts, please try again after some time." This improvement enhances the user experience by providing clear feedback and guidance in such scenarios. (RAR-2113)
  • Improved the behavior when filtering for Users, Apps, Status, and Severity on the Logs and Traces screens. Now, when filtering for a specific value on those properties, the respective column will no longer be hidden on the results list. (RDLOT-1957)
  • Fixed an issue that caused the Logs screen to not load when the user didn’t have permission to view the user’s information. (RDLOT-2005)
  • Added Sleep system action. (RTAFB-6769)
  • Added exception message when the output parameters aggregated size from a custom code action is too large (> 5.5 MB). (RTAFB-6782)

Bug Fixing

  • Resolved an issue that may have prevent administrator registration during the establishment of new tenants in recently expanded regions. (RDPIDT-1420)
  • Fixed a concurrency problem that caused the publication of an External Library to fail (OS-ELGW-50005). (RTAFB-6816)
  • Improved error message (previously OS-ELG-MODL-50010) when uploading an External Library package containing files that cannot be uncompressed. (RTAFB-6817)
  • Improved error message (previously OS-ELG-MODL-50010) when uploading an External Library package that was compressed using an unsupported method. (RTAFB-6823)
