OutSystems Developer Cloud 2023-06-16

Released on Jun 16, 2023

Bug Fixing

  • Fixed an issue that would randomly present the file size error even when the uploaded file was within the supported size limit. (RDLOT-1905)
  • Fixed an issue when using the drag-and-drop feature to upload files when creating an external library. This problem caused the creation of the external library to fail with the error code OS-ELG-40004. (RDLOT-1909)
  • Fixed a bug that caused External Library Actions names clashing with Structures names, resulting in a compilation error (OS-ELGW-50014) while publishing. (RTAFB-6784)
