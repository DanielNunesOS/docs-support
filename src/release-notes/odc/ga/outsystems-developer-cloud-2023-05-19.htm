OutSystems Developer Cloud 2023-05-19

Released on May 19, 2023

New

  • Apps - Added an informative message on the consumers and producers tabs under the Library detail page to clarify the information presented on those lists. (RDLOT-1556)
  • Apps - The user info card in the apps and library detail screens now displays the user name instead of the email. (RDLOT-1760)
  • Apps - The producers' tab under the library detail page is now hidden for External libraries. (RDLOT-1787)

Bug Fixing

  • Apps - Fixed a bug on the mobile distribution page, where downloading an iOS package from an iPad with a higher resolution would download the installer file instead of performing the installation of the app directly on the device. (RDLOT-1457)
  • Apps - Fixed an issue where the list of timers in app configurations would occasionally fail to display the timers' information when the accordion was opened too fast. (RDLOT-1724)
