OutSystems Developer Cloud 2023-06-05

Released on Jun 05, 2023

New

  • Forge - New approval e-mail template added for when a submitted asset it’s approved. (RMKPT-2356)
  • Forge - Users can now find the assets they installed from Forge more quickly, on the My Assets page, just by hiding the pre-installed assets. (RMKPT-2358)
  • Forge - Added the mandatory icon as a prerequisite to the submit journey to promote better identification of the assets. (RMKPT-2395)
  • Forge - The user identity of who submitted an asset to Forge is now visible to every user inside the producer organization. (RMKPT-2398)

Bug Fixing

  • Fixed an issue on the Submit to Forge Search - The “no results” message was displayed when searching for an asset that exists. This was happening after the search bar been cleared and the page number changed. (RMKPT-2428)
  • Fixed an issue that caused a publication error (OS-ELGW-50014) of an External Library generated from code that implemented the Library’s .NET Interface explicitly. (RTAFB-6745)
