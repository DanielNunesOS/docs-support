OutSystems Developer Cloud 2023-11-29

Released on Nov 29, 2023

New

  • If you open the details of a Connection’s entity in the ODC Portal and it has changed since the last time it was saved, the Portal will show you a list of changes. (RCNT-5627)
  • It’s now possible to create connections to SAP on the ODC Portal. These connections use the OData protocol and only allow read operations, for now. (RCNT-6739)

Bug Fixing

  • Fixed an issue on the ODC Portal’s Connections: when there was an error getting entity attributes, the Portal didn’t display an error message. (RCNT-6720)
