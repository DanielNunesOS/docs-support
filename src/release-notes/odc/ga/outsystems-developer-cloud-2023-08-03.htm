OutSystems Developer Cloud 2023-08-03

Released on Aug 03, 2023

New

  • Improved error messages when there is problem sending an email. Exception messages (previously 'Something went wrong on our side.') now include more information about the error. (RTAFB-6856)
  • Improved error message of JWT_ReadToken action from the Security library when the token provided is not well formatted. Exception messages (previously 'Something went wrong on our side.') now includes the original exception type and message. (RTAFB-6935)

Bug Fixing

  • Fixed an issue where the overview screen was not showing revision details for some apps in the development stage. (RDEL-1760)
