OutSystems Developer Cloud 2024-08-08

Released on Aug 08, 2024

New

  • Improve session stability and performance under low network connectivity conditions. (RAR-2755)
  • For SAP OData integrations, the "Linkedto" attributes will now have 'RecordList' as the data type instead of 'Record' in the ODC Portal. In ODC Studio, these attributes will continue to have 'Text' as the data type, as before. There are no breaking changes. (RDCIST-2211)

Bug Fixing

  • Fix Exception propagation across Service Action invocations. (RRCT-5480)
