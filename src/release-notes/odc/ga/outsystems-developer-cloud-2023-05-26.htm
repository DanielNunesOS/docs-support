OutSystems Developer Cloud 2023-05-26

Released on May 26, 2023

New

  • [ODC Portal] Added information about the app version to the Overview screen (RDEL-1446)
  • [ODC Portal] Timestamps in the Overview screen are now relative, so they are more meaningful. You may still hover it to see the full absolute timestamp. (RDEL-1447)
  • [ODC Portal] Added more contextual information to the Overview screen, showing the full deployment timestamp and the user responsible for the deployment. (RDEL-1452)
  • [ODC Portal] Added a manual refresh mechanism to the Overview screen, also letting the user know how long ago the information displayed was loaded (RDEL-1453)
  • [ODC Portal] Added pagination to the Overview screen (RDEL-1608)
  • Apps - The user info in the mobile distribution tab under the App detail screen now displays the user name instead of the email. (RDLOT-1829)
