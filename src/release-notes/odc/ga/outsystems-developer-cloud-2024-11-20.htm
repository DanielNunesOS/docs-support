OutSystems Developer Cloud 2024-11-20

Released on Nov 20, 2024

New

  • ODC Applications are now using a JavaScript bundler instead of RequireJS for a faster, more efficient bundling and improved future compatibility. (RAR-2480)
  • Workflow Editor: Allow to assign one or more Public Roles to a Human Activity
    • Is possible while creating an end-user role in ODC Studio to set it as the "Public" then the role can be shared across Apps and Workflows.
    • A new built-in function to check end user roles can is available.
    • Is possible to build UI components using workflow entities (taskbox).
    (RDODCT-2749)
  • Workflow Editor: Conditional Start and Terminate Nodes
    • Workflow editor allows to create Conditional start Nodes, to reflect the start of certain activities in case a certain event is triggered.
    • It allows as well to create Terminate nodes, which stopped everything that is running or not started when a certain activity is reached.
    (RDODCT-2751)
  • Workflow Editor: Allow to export/import a Workflow OML
    • Workflow Editor provides the option to export a Workflow (OML) from Workflow Editor
    • It allows as well to import a Workflow (OML) from ODC Portal Â» Workflows Menu
    (RDODCT-2752)
  • Workflow Editor: Wait Nodes
    • Workflow Editor allows the possibility to model workflows with Wait Nodes, which allows to pause activities until a certain application event occurs.
    (RDODCT-2753)
