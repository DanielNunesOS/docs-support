OutSystems Developer Cloud 2023-07-19

Released on Jul 19, 2023

New

  • Added the ability to modify the mapped scopes when configuring an Identity Provider through the ODC Portal, to provide more flexibility for Providers who require specific scopes to be used during authentication. (RDPP-1258)
  • Added the ability to modify the additional endpoints when configuring an Identity Provider through the ODC Portal, to provide more flexibility for Providers who do not return all the required information from the well-known endpoint. (RDPP-1273)

Bug Fixing

  • Fixed a visual issue in the details of a Identity Provider where the assignments could be cut off. (RDPP-1237)
  • Fixed an issue on the Identity providers list where after deleting an Identity provider, the search filter would not be applied when refreshing the list (RDPP-1280)
