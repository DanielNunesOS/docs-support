OutSystems Developer Cloud 2024-05-31

Released on May 31, 2024

New

  • When submitting external logic libraries to Forge, you now have the ability to specify the public repository URL where the source code used for that library, in case you want to share it with the community to foster trust and transparency. (RPOR-20138)
  • Monthly Security Update (May 2024): Updated application container base image, including security fixes. Check Microsoft .NET 8.0.5 Release for more details: https://github.com/dotnet/core/blob/main/release-notes/8.0/8.0.5/8.0.5.md (RRCT-5769)

Bug Fixing

  • Fixes a bug where links to screens with arguments in lists and tables wouldn't update if only the arguments changed. (RAR-2687)
