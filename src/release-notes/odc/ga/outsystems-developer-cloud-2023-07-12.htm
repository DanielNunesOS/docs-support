OutSystems Developer Cloud 2023-07-12

Released on Jul 12, 2023

New

  • The application detail screen on the ODC Portal now presents the publish date and user for the development stage and presents the deployment date and user for other stages. (RDLOT-1937)

Bug Fixing

  • Fixed an issue in the mobile distribution tab, on the app detail screen, that would present the ‘Create package' button as enabled when the user didn’t have permission to perform the operation. The user would still be blocked from creating a mobile package, but now the validation is performed earlier, and the button will be disabled in this situation. (RDLOT-1900)
  • Fixed an issue in the mobile distribution tab, on the app detail screen, that would expand/collapse the mobile package information when clicking on the download packages or download logs buttons. (RDLOT-2001)
  • Fixed an issue on the ODC Portal that would block the users from changing app configurations when there was an error during 1CP. (RDLOT-2014)
  • Fixed the link that redirects to the MABS documentation when creating a new mobile package. (RDLOT-2018)
