OutSystems Developer Cloud 2023-12-11

Released on Dec 11, 2023

New

  • Improved the experience in the Subscription page on ODC Portal when the license limits are exceeded, so users understand the potential impacts, if any. (RDEL-1950)
  • Added detailed information on the Subscriptions page on ODC Portal about the number of AOs being used on each stage. (RDEL-1951)

Bug Fixing

  • Fixed an issue in the dependency analysis report, that could cause the console to not show information when it returns Name collision mismatch. (RDEL-1929) (RDEL-1929)
  • Fixed an issue in the Subscription on ODC Portal that could cause the number of allowed end-users (external or internal) to be incorrectly displayed as unlimited when it should be zero. (RDEL-1956)
  • Fixed an issue in the Subscription on ODC Portal where the support level could be incorrectly displayed as 24x7 support instead of the actual subscribed 8x5 support. (RDEL-1959)
  • Fixed the character limit on the error code field when submitting a new Support case. (RMKPT-2672)
  • Fixed an issue in BinaryDataToText where the byte order mark was not removed when performing the conversion. (RTAFB-7420)
