OutSystems Developer Cloud 2023-07-05

Released on Jul 05, 2023

New

  • It’s now easier to see which Forge assets are outdated. On the My Assets page, the assets installed from Forge with updates appear at the top of the list. (RMKPT-1993)
  • Improved copy of the tooltips in the Forge Submit journey, giving a better guidance on the Overview step. (RMKPT-2404)
  • Improved search experience by resetting the pagination of each Forge page on login in. (RMKPT-2451)
  • It’s now possible to submit App Templates to Forge, allowing consumers to install them. A version number will be added to the revision submitted. (RMKPT-2465)

Bug Fixing

  • Fixed a pagination issue when the pre-installed assets toggle was on in the My Assets screen of Forge. (RMKPT-2462)
