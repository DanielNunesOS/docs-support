OutSystems Developer Cloud 2023-10-06

Released on Oct 06, 2023

New

  • Added an error message when an external library action fails because the custom code exceeded the allowed time limit (95s). (RTAFB-7219)

Bug Fixing

  • Fixed an issue in the user’s invite page that could cause the filters for end-user roles to disappear. (RDPP-1420)
  • Fixed a bug that caused a {{Foreign Key constraint violation database exception}} to be raised during the execution of the {{CreateOrUpdateSome Entity Action}} when a {{non-mandatory Foreign Key attribute}} was set to an empty value. The {{CreateOrUpdateSome Entity Action}} now handles empty {{non-mandatory Foreign Key}} values correctly, without violating any {{Foreign Key constraints}}. (RPLAT-2078)
  • Fixed error in External Libraries that contain custom code that fills in action’s outputs with null values. (RTAFB-6864)
