OutSystems Developer Cloud 2023-12-01

Released on Dec 01, 2023

New

  • Now, on the configurations details pages it is possible to check the value when typing in password fields. (RDLOT-2134)
  • Fixed an issue on the ODC Portal where sometimes while applying configurations consecutively they were not being applied correctly. (RDLOT-2416)
  • New icons added for the Libraries provided by OutSystems (Binary Data, Date Time, HTTP, Math, Sanitization, Security, Text, Text Dictionary, URL, XML and Zip). (RTAFB-6850)

Bug Fixing

  • Fixed an issue on the ODC Portal where the libraryâ€™s release notes were not showing some characters correctly on the version history tab under the application details screen. (RDLOT-2383)
  • Fixed an issue on the ODC Portal where no feedback message was sent after applying the configurations successfully on the application details screen. (RDLOT-2410)
  • Fixed an issue on the ODC Portal where the option to configure test emails on the application details page was not available. (RDLOT-2445)
  • Fixed a bug that caused a {{Npgsql exception}} to be raised during the execution of the {{CreateOrUpdateSome Entity Action}} when a {{non-mandatory Foreign Key attribute}} was set to an empty value. The {{CreateOrUpdateSome Entity Action}} now handles empty {{non-mandatory Foreign Key}} values correctly, without raising any {{Npgsql exception}}. (RPLAT-2138)
  • Created a new error (code OS-ELG-MODL-05030) when uploading an External Library with the unsupported scenario of using a custom collection class (extending IEnumerable) as input or output of an action (previously OS-ELG-MODL-50010) (RTAFB-7407)
