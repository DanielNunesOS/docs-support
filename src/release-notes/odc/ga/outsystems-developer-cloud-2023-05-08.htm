OutSystems Developer Cloud 2023-05-08

Released on May 08, 2023

New

  • ODC Portal Users: Improved the experience for user details. The “End user access” section has now search filters globally that will apply to both *Roles inherited from groups* and to *Roles* sections. (RDPP-1151)

Bug Fixing

  • End-user groups: Fixed an issue that caused an end-user group that contained uses associated with invited status to properly load the users' information on the end-user group detail page. (RDPP-1174)
