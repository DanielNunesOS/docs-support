OutSystems Developer Cloud 2024-09-24

Released on Sep 24, 2024

New

  • You can now import and export workflows. To import a workflow, go to the Workflows area, select "Import" and upload a workflow OML file. To export a workflow, open the workflow details and, under the "Revisions" section, select "Export as OML" for the revision you want to export. (RDLOT-3201)
