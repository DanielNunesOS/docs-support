OutSystems Developer Cloud 2024-12-03

Released on Dec 03, 2024

New

  • In ODC Logs, there is a new tab specific to Workflows where you can search, analyze, and monitor the activities log. You can filter the logs by multiple criteria: Workflow, Instance ID, Activity, user and time. It is also possible to search the workflow logs by keywords present in the log message (RDASS-2402)
