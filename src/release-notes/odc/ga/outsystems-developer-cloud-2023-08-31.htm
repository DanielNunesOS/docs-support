OutSystems Developer Cloud 2023-08-31

Released on Aug 31, 2023

New

  • Changed the way Connection Entities are loaded in the ODC Portal. The screen should load a bit faster now. (RCNT-6372)

Bug Fixing

  • Fixed an issue in the ODC portal where stages could overflow the Edit Connection screen. (RCNT-6248)
  • Now, the input for Connection descriptions, on the ODC Portal, only takes 250 characters. Previously it accepted more characters, which could cause errors in saving connections. (RCNT-6269)
  • Fixed an issue where the ODC Portal would not warn users when the name they input for a Connection was already in use. (RCNT-6324)
