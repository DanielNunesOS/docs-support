OutSystems Developer Cloud 2023-07-24

Released on Jul 24, 2023

Bug Fixing

  • Fixed an issue that could happen when activating a Secure Gateway that showed a message warning the user that it was not working as expected, even though the Secure Gateway was correctly activated. (RDEL-1643)
  • Fixed an issue in the Secure Gateways console that could trigger a 403 Forbidden error message trying to get the status of a secure gateway for which the user did not have permissions for. (RDEL-1691)
