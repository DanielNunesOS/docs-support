OutSystems Developer Cloud 2024-09-23

Released on Sep 23, 2024

New

  • Improved navigation performance between pages provided by different apps when Client-Side traces are enabled. (RAR-2815)
  • You can now start a deployment for an app directly on the App Detail screen. Select this option under the "more options" menu and you'll be redirected to the deployment wizard, where you can choose the specific revision you want to deploy. (RDLOT-3021)
  • You can now start a deployment for a workflow directly on the Workflow Detail screen. Selecting this option under the "more options" menu will take you straight to the deployment wizard, where you can choose the specific revision you want to deploy. (RDLOT-3076)

Bug Fixing

  • Fix bug where when Download nodes are used with If statements, it can cause all branches to execute (RAR-2836)
  • Fixed an issue that caused logs to be spammed when a user did not have the correct role. (RAR-2877)
  • Fixed an issue with the validation of Form widgets, which failed when using a button placed inside a block or outside the Form widget. (RAR-2891)
