OutSystems Developer Cloud 2023-08-11

Released on Aug 11, 2023

New

  • For external entities, fixed a bug that prevented developers to join entities with attributes that have the same name but in different cases Example: EntityA.Id = EntityB.id. (RDDFC-1717)
  • NAOS.6 now returns the state of the MABS Versions (RNMT-6155)

Bug Fixing

  • Fixed an issue, on the ODC Portal, where the “Save” button on a connection's “Select Entities” screen remained disabled after a mandatory column was added to a table. (RCNT-6363)
  • Enhances trace propagation on running jobs (RPLAT-1980)
