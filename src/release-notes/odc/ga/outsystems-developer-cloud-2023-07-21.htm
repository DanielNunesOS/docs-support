OutSystems Developer Cloud 2023-07-21

Released on Jul 21, 2023

New

  • You can now sort Forge all assets list by the ones most installed, so that you may know which are the most popular ones. (RMKPT-2491)
  • Improved error message of JWT_ReadToken action from the Security library when there are validation errors. Exception message (OS-SLIB-SEC-50011) now includes the original exception type and message. (RTAFB-6841)

Bug Fixing

  • Resolved an issue causing errors when having JS scripts with names ending in "outsystems" in applications or libraries. (RBST-1561)
  • Fixed an issue in Forge that caused the My Assets page to be continually refreshed unnecessarily when loading the page. (RMKPT-2531)
  • Fixed an issue in Forge on the My Assets page that sometimes caused the assets on the Installed from Forge tab to not appear when navigating from the Submitted to Forge, (RMKPT-2550)
  • Fixed an issue in Forge on the My Assets page that caused the pre-installed assets to be displayed when clearing the search keyword, ignoring the pre-installed assets toggle. (RMKPT-2551)
