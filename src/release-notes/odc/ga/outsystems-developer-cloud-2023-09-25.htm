OutSystems Developer Cloud 2023-09-25

Released on Sep 25, 2023

New

  • Updated the deployment detail page on ODC Portal to show the user name and picture (if available) instead of just the email. to make it easier to identify the person responsible for the deployment. (RDEL-1664)
  • Updated the deployments list page on ODC Portal to show the user name and picture (if available) instead of just the email. to make it easier to identify the person responsible for each deployment. (RDEL-1665)
  • Updated the revisions list page of the deployment wizard on ODC Portal to show the user name and picture (if available) instead of just the email. to make it easier to identify the person responsible for the deployment of each revision. (RDEL-1666)
  • Updated the stages overview page on ODC Portal to show the user name and picture (if available) instead of just the email. to make it easier to identify the person responsible for the deployment. (RDEL-1667)
  • Updated some of the messages in the Impact Analysis report during the Deployment operation, to make them easier to understand. (RDEL-1824)
  • Added a link in the Impact Analysis step of the deployment process to the public documentation where you can learn more about the inconsistencies found during the deployment (RDEL-1826)

Bug Fixing

  • Fixed an issue in the deployments list screen on ODC Portal that could manifest itself when a new stage is added to the organization, after apps had already been deployed to any non-development stage. (RDEL-1680)
  • Fixed an issue in the Overview page on ODC Portal that would cause the deployed revisions of a renamed app to not be correctly displayed when searching by that app. (RDEL-1844)
