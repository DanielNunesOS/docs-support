OutSystems Developer Cloud 2023-03-28

Released on Mar 28, 2023

New

  • Improved the QR Code size and quality on the mobile distribution tab under the App details page. (RDLOT-1561)
  • Forge - Improvement : Label “Submitted to Forge” or “Pending approval” continues to appear for the producer of the asset even if there is already a new revision/version of the application deployed to production. (RDFNO-192)
  • Forge - Users can now select the sort by option in the All assets page, being able to sort by most recent or A → Z. (RMKPT-2240)
  • Forge - Improved Submit journey
    • Merge the Basic information step with the Overview step.
    • Display of asset version information in the Documentation step.
    • New Review & submit step that will allow the preview of the asset detail page before final submission.
    (RPOR-10763)

Bug Fixing

  • Forge - Fixed an issue in the submit journey - when an image was being uploaded on the documentation, in the fullscreen mode, the toolbar of the editor would overlap with the submit confirmation popup. (RMKPT-2323)
  • Forge - Fixed an issue where the size of images on the documentation could change after saving. (RMKPT-2325)
