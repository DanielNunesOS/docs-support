OutSystems Developer Cloud 2024-07-22

Released on Jul 22, 2024

Bug Fixing

  • Fixed an issue where creating an Android package always required uploading a Keystore file, even if one was already configured from a previous package. (RDLOT-2952)
  • Fixed an issue in the version tag validation when releasing a library, which in some situations displayed an error, even when the new version was higher than the current one. (RDLOT-3027)
  • Fixed issue that caused username to appear as Unknown User on Portal. (RDPIDT-1113)
  • Resolved an issue that may have prevent administrator registration during the establishment of new tenants in recently expanded regions. (RDPIDT-1420)
