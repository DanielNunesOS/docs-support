OutSystems Developer Cloud 2023-06-23

Released on Jun 23, 2023

New

  • Had to add a watcher for environments on timermanager, that checks if the necessary service account exists, and creates it if it's not there. (RRCT-5043)
  • OPA policy solved (RRCT-5046)
  • Increased the maximum amount of RAM memory available when executing the .NET Code of an External Library from 256 MB to 512 MB. This change also increases the CPU available, resulting in lower execution times. (RTAFB-6757)

Bug Fixing

  • Fixed an issue that caused a publication error (OS-ELGW-50014) of an External Library generated from code that implemented the Libraryâ€™s .NET Interface explicitly. (RTAFB-6745)
  • Fixed a bug that caused External Library Actions names clashing with Structures names, resulting in a compilation error (OS-ELGW-50014) while publishing. (RTAFB-6784)
  • Improved error message (OS-ELG-MODL-5024) when uploading an External Library package that uses structures not decorated with the OSStructure attribute. (RTAFB-6786)
