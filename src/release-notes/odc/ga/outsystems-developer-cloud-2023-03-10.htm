OutSystems Developer Cloud 2023-03-10

Released on Mar 10, 2023

New

  • Forge - Improved behavior of highlighting missing mandatory fields while submitting or editing an asset. (RMKPT-2142)
  • Forge - Tooltips copy reviewed on the first step of the submit journey, giving better guidance on having the asset ready for submission. (RMKPT-2162)
  • Forge - Libraries version number is now visible in the dependency pop-up in the submit journey. (RMKPT-2298)
  • Forge - System libraries are now identified as asset dependencies in Forge when applicable. (RMKPT-2299)

Bug Fixing

  • Forge - Fixed an issue where the assets list in the My Assets page was not behaving correctly when moving between tabs while searching for a keyword. (RMKPT-2213)
  • Forge - Fixed an issue where the assets list in the My assets page was loaded empty if the user accessed it directly via URL, after login in. (RMKPT-2215)
  • Forge - Fixed an issue where the last image of the documentation was not being saved. (RMKPT-2297)
