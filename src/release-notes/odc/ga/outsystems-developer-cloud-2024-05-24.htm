OutSystems Developer Cloud 2024-05-24

Released on May 24, 2024

New

  • You can now sort the apps list on the Overview screen by the date the apps were last deployed on each stage. (RDLOT-2920)

Bug Fixing

  • Fixed an issue in the Oracle connector that caused a wrong data type mapping for foreign keys when the associated entity was not selected. Foreign keys should have the Identifier data type (Integer, Long Integer, or Text) when the associated entity is not selected in the Connections console. (RDCIST-1947)
