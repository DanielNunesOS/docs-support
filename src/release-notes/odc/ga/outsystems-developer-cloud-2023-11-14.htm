OutSystems Developer Cloud 2023-11-14

Released on Nov 14, 2023

New

  • Improved the user invite experience, in the scenarios where the user is invited (receives the ODC invite email) but not all the user access configurations were correctly saved. You will now be able to review the user’s access and correct any missing configurations. (RDPP-1184)
  • Improved the end-user groups information in the user’s profile page by adding the total number of roles and stage that each group is assigned to. (RDPP-1298)
  • When creating or editing an organization role on ODC Portal, you will now remain on the role’s detail page so you can review the changes made. This experience is also more aligned with the rest of ODC Portal. (RDPP-1346)
  • Improved the experience when managing the scopes of an Identity Provider to make it explicit that you need at least one scope configured. (RDPP-1350)
  • Updated the user detail screen on ODC Portal to show 10 roles per page in the organization access and end-user access tabs, providing you more information in a single view. (RDPP-1410)
  • Updated the user invite screen on ODC Portal to show 10 roles per page in the organization access and end-user access sections, providing you more information in a single view. (RDPP-1411)

Bug Fixing

  • Fixed an issue where the same message could be shown twice in the end-user profile page. (RDPP-1318)
  • Fixed an issue that could cause the user’s profile page to be incorrectly rendered if the user’s name was too big. (RDPP-1460)
