OutSystems Developer Cloud 2023-11-20

Released on Nov 20, 2023

Bug Fixing

  • Fixed an issue during the Impact Analysis step of the deployment journey, where the user could become blocked in case an unexpected error occurred during the analysis. (RDEL-1885)
  • Fixed an issue when creating a new Identity Provider on ODC Portal using an accelerator that would prevent the user from successfully creating it due to an incorrect validation of the mapped scopes. (RDPP-1529)
