OutSystems Developer Cloud 2024-07-11

Released on Jul 11, 2024

New

  • For external entities, in the ODC Studio, the attributes are now sorted in the same order as they appear in the external system. For existing connections, it is recommended to "Refresh entities list" and refresh dependencies in ODC Studio. (RDCIST-2094)

Bug Fixing

  • Fixes an issue that caused applications running on an iOS device prior to version 16.4 to fail to load and display solely a blank screen. (RAR-2745)
  • Fixes an issue that caused Web Applications with a Splash Screen set to fail to load with a `No cached manifest found. It's not possible to rollback to a previous version.` error. (RAR-2749)
