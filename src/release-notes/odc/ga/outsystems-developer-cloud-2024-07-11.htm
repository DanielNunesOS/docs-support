OutSystems Developer Cloud 2024-07-11

Released on Jul 11, 2024

New

  • Service is promoted to dev ring -3 and apis are accessed successfully (RDASS-465)
  • New icons added for the Libraries provided by OutSystems (Binary Data, Date Time, HTTP, Math, Sanitization, Security, Text, Text Dictionary, URL, XML and Zip). (RTAFB-6850)
  • Improved the error messages on System Extensions to reduce the amount of times an error gets hidden by the message _"Something went wrong on our side"_, providing developers with more information about the errors they may encounter. System Extensions updated: * BinaryData * HTTP * Math * Security * URL (RTAFB-7251)

Bug Fixing

  • Fixed an issue in BinaryDataToText where the byte order mark was not removed when performing the conversion. (RTAFB-7420)
